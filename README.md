## RF (Requisitos funcionais)

- [ ] O usuário deve poder se registrar
- [ ] O usuário deve poder se autenticar
- [ ] O usuário deve poder acessar seu perfil
- [ ] O usuário deve poder listar por quadras próximas a ele
- [ ] O usuário deve poder pesquisar por quadras por endereço

## RN (Regras de negócio)

- [ ] Não pode haver dois usuários com o mesmo email
- [ ] A senha deve ser armazenada como um hash da senha original
